# Changelog

Todas las versiones publicadas de este proyecto se documentan aquí.

## V1.0.0 - 2026-03-28

- formaliza el versionado semántico con archivo `VERSION`
- agrega licencia `Apache License 2.0`
- documenta el proyecto, dependencias, operación y flujo de releases
- agrega automatización de release en GitHub Actions para cada push a `main`
- muestra la versión y el enlace del repositorio dentro de la app
- mejora el escaneo LAN para distinguir cámaras web compatibles frente a dispositivos con protocolo propietario
- evita falsos positivos de router y de servicios tipo AirTunes
- añade detección diagnóstica de dispositivos `Tuya/Smart Life` con puertos propietarios
